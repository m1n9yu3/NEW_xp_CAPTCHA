# xp_CAPTCHA(白嫖版)

### m1n9yu3 个人升级部分

flask 多线程， 否则原始http 请求库 会阻塞页面， 影响到验证码识别


打包成 docker 镜像

```shell
docker pull m1n9yu3/centos-xp-captcha:1.0
```


端口映射 8899 出去


进入 容器， 运行 

```shell
python3 server2.py
```

后台运行
```shell
nohup python3 server2.py
```

### 安装

需要python3 小于3.7的版本

安装 `muggle_ocr` 模块（大概400M左右）
```
python3 -m pip install -i http://mirrors.aliyun.com/pypi/simple/ --trusted-host mirrors.aliyun.com muggle-ocr
```
